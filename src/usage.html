Everything can be done from the index.html page on the website or from going to doaction.php passing in the action and the relevant parameters in the url. Example: doaction.php?action=register&username=test&password=test

Notes

  • setguildmaster doesn't remove username2 from members, if one wants to do that, kick username2 from the guild, then set them as the guildmaster
  • leaving as guildmaster also disbands the guild (kicking everyone from the guild), so if you're the guildmaster and want to leave, but don't want to end the guild, setguildmaster to anyone you want to give the guild to before you leave
  • if one wants to make it so that members of a guild at a certain rank can have guildmaster priviledges, just make a guildmaster user and then give the password of the guildmaster user to users that get to that rank

Action - Description [parameters]

Add:

  • register - register [username, password]
  • request - request to be friends with username2 [username, password, username2]
  • makeguild - starts a guild as guildmaster [username, password, guildname]
  • invite - invite username2 to guild (only guildmaster can) [username, password, guildname, username2]

Set:

  • setonline - sets a user's online status (0 = offline, 1 = online) [username, password, online]
  • setguildmaster - makes user the guildmaster (only guildmaster can do this) [username, password, guildname, username2]
  • setrank - set the rank of a guild member (must be guildmaster) [username, password, guildname, username2, rank]

Accept:

  • acceptfriend - accept friend request from username2 [username, password, username2]
  • acceptguild - accept guild invite from guildname [username, password, guildname]

View:

  • isonline - checks if a friend is online (must be friends) (0 = offline, 1 = online) [username, password, username2]
  • listfriends - lists the friends of a user [username, password]
  • viewmyrequests - list a user's friend requests to other users [username, password]
  • viewrequests - list a user's friend requests from other users [username, password]
  • viewmemebers - list the ranks of all members of a guild (must be in the guild) [username, password, guildname]
  • viewmyinvites - list the guild invites a user has received [username, password]
  • viewguildsinvites - list the invites a guild has sent out (must be a member of that guild) [username, password, guildname]

Remove:

  • removefriend - removes a friend from a user's friends list [username, password, username2]
  • leave - leave a guild [username, password, guildname]
  • kick - kick a member from a guild (must be guildmaster) [username, password, guildname, username2]
  • declinefriend - removes a friend request from a user [username, password, username2]
  • declineinvite - removes a guild invite from a guild [username, password, guildname]
  • removerequest - removes a friend request sent to a user (must be guildmaster) [username, password, username2]
  • removeinvite - removes a guild invite sent to a user (must be guildmaster) [username, password, guildname, username2]
